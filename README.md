# gestion-hospitalaria-cpp

------------------Optimización de Rutas de Ambulancias------------------------

Este módulo agrega al sistema la capacidad de calcular la ruta más corta entre dos puntos de la ciudad, usando un grafo no dirigido y el algoritmo Breadth-First Search (BFS).

Funcionalidad:

- Representación del mapa mediante una lista de adyacencia.

- Cálculo de rutas óptimas con BFS (O(V+E)).

- Validación de nodos ingresados por el usuario.
